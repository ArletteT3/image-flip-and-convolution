## Project Overview
This project demonstrates two basic image processing operations implemented from scratch using Python and NumPy:

1. Vertical image flipping
2. 2D convolution using a custom 3x3 kernel

The notebook allows users to upload an image, process it, and visualize the results using Matplotlib.

## Features
- Upload any image file (PNG, JPG)
- Vertically flip the image using NumPy array slicing
- Apply a custom convolution kernel
- Display original, flipped, and processed images

## Technologies Used
- Python
- NumPy
- Matplotlib
- Google Colab

## Example Kernel
Sharpen filter used:

[ 0 -1  0  
 -1  5 -1  
  0 -1  0 ]
